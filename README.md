# أعراف Legal OS — المنصة القانونية الذكية

منصة قانونية ذكية متكاملة لشركة أعراف للمحاماة والاستشارات القانونية.

## المميزات

- **المساعد القانوني AI** — بحث قانوني عميق في الأنظمة السعودية الرسمية مع مصادر موثقة
- **مولّد العقود الذكي** — إنشاء 8 أنواع عقود متوافقة مع الأنظمة السعودية
- **محلل مخاطر العقود** — تحليل بنود العقود واكتشاف المخاطر القانونية
- **المكتبة القانونية** — مرجع شامل للأنظمة واللوائح السعودية
- **استشارات المحامين** — التواصل مع محامين سعوديين مختصين
- **سجل الباقة** — متابعة استهلاك الاشتراك الشهري

## البنية التقنية

```
├── index.html          # الصفحة الرئيسية
├── style.css           # التصميم الكامل
├── script.js           # المنطق البرمجي (الواجهة)
├── logo.jpg            # شعار الشركة
├── api/
│   └── ask.js          # الخادم (Vercel Serverless)
├── package.json        # المكتبات
├── vercel.json         # إعدادات Vercel
└── .gitignore
```

## النشر على Vercel

1. ارفع المشروع على GitHub
2. اربطه بـ [Vercel](https://vercel.com)
3. أضف المتغيرات البيئية:
   - `OPENAI_API_KEY` — مفتاح OpenAI API
   - `SERPER_API_KEY` — مفتاح Serper API للبحث
4. انشر المشروع

## التقنيات المستخدمة

- Vanilla HTML/CSS/JS (بدون فريمورك)
- Vercel Serverless Functions (Node.js)
- OpenAI GPT-4.1 API
- Serper Google Search API
- Cheerio (استخراج محتوى الصفحات)
- pdf-parse (قراءة ملفات PDF)
